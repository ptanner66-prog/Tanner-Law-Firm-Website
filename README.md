# Tanner Law Firm Website

A professional, responsive website for Tanner Law Firm built with HTML5, CSS3, and vanilla JavaScript.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design appropriate for a law firm
- **Interactive Elements**: Smooth animations, mobile navigation, and form validation
- **Multiple Pages**: Home, About, Practice Areas, Attorneys, and Contact pages
- **SEO Optimized**: Semantic HTML with proper meta descriptions

## Pages

- `index.html` - Home page with hero section, practice areas overview, and testimonials
- `about.html` - About the firm, values, and why choose us
- `practice-areas.html` - Detailed practice area pages (Personal Injury, Family Law, Criminal Defense, Business Law)
- `attorneys.html` - Attorney profiles and credentials
- `contact.html` - Contact form and office information

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - it's a static site

## Customization

- **Colors**: Edit CSS custom properties in `css/styles.css` under `:root`
- **Content**: Update text content directly in HTML files
- **Images**: Replace placeholder divs with actual images in the `images` folder
